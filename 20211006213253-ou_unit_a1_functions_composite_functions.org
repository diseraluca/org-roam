\begin{tcolorbox}[title=Composite Function]
Let $f : A \rightarrow B$ and $g : B \rightarrow C$ be the two functions such that the domain of $g$ is the same set as the codomain of $f$.

Then the \textbf{composite function} $g \circ f$ is given by

\begin{align*}
g \circ f : &A \rightarrow C \\
        &x \mapsto g(f(X))
\end{align*}

The composite function has:

\begin{itemize}
\item domain $\{x \in A : f(x) \in C\}$
\item codomain D
\
\end{itemize}
\end{tcolorbox}

Imagine a function as a black-box taking an input and pushing out an output.

Then, the composition of those two function is a black-box that takes an input links the two original function.

\begin{tcolorbox}[title=Composite Function With Different Domain And Codomain]
Let $f : A \rightarrow B$ and $g : C \rightarrow D$ be any two functions.

Then the \textbf{composite function} $g \circ f$ is given by has:

\begin{itemize}
\item domain $\{x \in A : f(x) \in C\}$
\item codomain D
\item rule $(g \circ f)(x) = g(f(x))$
\end{itemize}
\end{tcolorbox}
