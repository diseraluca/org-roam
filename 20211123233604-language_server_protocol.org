#+title: Language Server Protocol

The Language Server Protocol is a [[id:2193ddf4-f5ec-4204-ab78-b8f6cdc772a7][JSON-RPC:2.0]]-based protocol that
helps with providing IDE like capabilities to source code editor, by
abstracting the language-specific processing to common interfaces.
