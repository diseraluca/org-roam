#+title: Language Server Protocol:3.16:Progress Reporting:ProgressParams:ProgressToken
#+filetags: :LSP:Tech:Languages:Tooling:Specification:Progress:Asynchronous:Token:

A [[id:18188d0a-f0bd-4e20-84c3-5291da9736c6][JSON:Type System:Number]] with no fractional part or a [[id:40b19f64-f8b5-49ec-9d7b-f095062801df][JSON:Type
System:String]] representing a task for whose progress is being
reported.

 + This is distinct from the ~id~ of a [[id:ad9a41bf-e8ca-4db9-b2dd-987786b21a22][JSON-RPC:2.0:Request]].

This is generated by either the Client or the Server and used to
recognize what is a progress reporting about.
