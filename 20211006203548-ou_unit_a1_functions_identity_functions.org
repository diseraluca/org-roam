\begin{tcolorbox}[title=The Identity Function]
The \textbf{identity function} on a set $A$ is the function

\begin{align*}
i_A : &A \rightarrow A \\
      &x \mapsto x
\end{align*}

\end{tcolorbox}

The *identity function* maps each element of the domain to itself.

For example, given the set \(A = \{0,1,2,3\}\), the identity function \(i_A\) is:

\begin{equation*}
i_A(0) = 0
\end{equation*}
\begin{equation*}
i_A(1) = 1
\end{equation*}

\begin{equation*}
i_A(2) = 2
\end{equation*}


\begin{equation*}
i_A(3) = 3
\end{equation*}
