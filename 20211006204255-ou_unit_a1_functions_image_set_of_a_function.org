#+LATEX_HEADER: \usepackage{tcolorbox}

\begin{tcolorbox}[title=Image]
Let $f : A \rightarrow B$ be a function. For any subset $S$ of $A$, the \textbf{image} of $S$ under $f$, denoted by $f(S)$, is the set:

\begin{equation*}
f(S) = \{f(x) : x \in S\}
\end{equation*}
\end{tcolorbox}

\begin{tcolorbox}[title=Image Set]
The \textbf{image set} of a function $f : A \rightarrow B$ is the set

\begin{equation*}
f(A) = \{f(x) : x \in A\}
\end{equation*}
\end{tcolorbox}

\begin{tcolorbox}[title=Surjective Function]
A function $f : A \rightarrow B$ is \textbf{onto} or \textbf{surjective} if $f(A) = B$
\end{tcolorbox}

    + In a *surjective function*, each element of the codomain is the image of at least one element of the domain.
